# ☁️ AI Cloud Cost Optimization System

## 🚀 Overview
An AI-powered cloud cost optimization system that detects anomalies, predicts future costs, and provides intelligent recommendations for enterprise cloud environments.

## 🎯 Features
- 📊 Interactive dashboard (Streamlit)
- 🔍 Filters by service
- 🚨 Anomaly detection (cost spikes)
- 🔥 Risk scoring (cost vs usage)
- 🤖 AI recommendation engine
- 🔮 Machine Learning cost prediction
- 📈 Trend charts and analytics
- 📂 Upload your own dataset

## 🧠 Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
